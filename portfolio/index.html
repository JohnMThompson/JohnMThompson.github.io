Home Office Temperature

  • June 2022

My first project using a Raspberry Pi. My goal was to experience gathering, ingesting, storing, refining, and presenting data along with all the DevOps work to run my applications on linux servers hosted by Digital Ocean. I configured a sensor to detect the temperature and humidity in my home office once per minute. Using Python, I wrote the results to MySQL and then delivered a dashboard with Metabase. The dashboard is now inactive but can still be viewed.
